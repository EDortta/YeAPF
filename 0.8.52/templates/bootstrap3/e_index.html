Obrigado por vir!

Este é apenas um pequeno sistema de exemplo integrando BootStrap3 e YeAPF.
Qualquer versão da serie 0.8.x do YeAPF deve poder ser utilizada aqui, mas estamos usando especificamente a 0.8.47
Incluí as fontes e o css na distruibuição porque na minha magra experiência, os servidores cdn mais atrapalham do que ajudam, em especial em aplicativos intramuros.
Uma das vantagens de trabalhar com este template é que pode ser fácilmente modificado com o Pingendo ou outros editores de BootStrap3
Abra o código e repare que cada uma das section tem um id e faz parte da classe tnTab.
Incluí um script de exemplo para alternar as abas clicando nas opções de menú logo acima.
Se quiser brincar, pode mudar de aba usando o id do menú, assim:  

sampleObj.openTabByClick(y$('ordens_op'));
